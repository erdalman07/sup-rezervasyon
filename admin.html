📋 Kurulum Adımları
1
SQL Şemasını Çalıştır
Supabase SQL Editor'ü aç → supabase-schema.sql dosyasının içeriğini yapıştır → RUN. Tablolar ve RLS politikaları otomatik oluşur.
2
Bağlantıyı Test Et
Yukarıdaki "🔌 Bağlantıyı Test Et" butonuna bas. ✅ Başarılı mesajı görürsen kurulum tamam!
3
Fiyat & IBAN Ayarla
Sol menüden 💰 Fiyat & Ayarlar'a git → IBAN, hesap adı, fiyatları gir → Kaydet. Bu bilgiler rezervasyon sayfasına otomatik yansır.
+
OpenWeather API (Hava Durumu)
1. openweathermap.org'a git → Ücretsiz hesap oluştur
2. API Keys bölümünden API anahtarını kopyala
3. rezervasyon.html dosyasında CONFIG.WEATHER_API_KEY satırına yapıştır
4. Ücretsiz plan günde 1000 istek hakkı verir (yeterli).
🚀
GitHub Pages ile Ücretsiz Yayınla
1. github.com'da ücretsiz hesap aç → Yeni repository oluştur
2. rezervasyon.html ve hero.jpg dosyalarını yükle
3. Settings > Pages > Branch: main → Save
4. URL: https://kullaniciadı.github.io/repoadi
